# CoffeeChatly

A coffee chat organizing and tracking app built for **MIT Sloan MBA Intro to Operations Management**.

CoffeeChatly helps you schedule, track, and follow up on coffee chats—whether with classmates, alumni, or recruiters—so you can stay on top of networking without losing the thread.

## Project context

- **Course:** MIT Sloan MBA – Introduction to Operations Management  
- **Purpose:** Organize and track coffee chats in one place  
- **MVP design:** User flow and UI are based on the [Figma prototype](https://www.figma.com/design/pBAPllz4rVDyDIsjsQXKul/User-flow-diagram-for-MVP).

## Tech stack

- **React** + **TypeScript**
- **Vite** for build and dev server
- **Tailwind CSS** for styling
- **React Router** for navigation
- **MUI** and **Radix UI** components

## Getting started

1. **Install dependencies**

   ```bash
   npm i
   ```

2. **Start the development server**

   ```bash
   npm run dev
   ```

3. **Build for production**

   ```bash
   npm run build
   ```
